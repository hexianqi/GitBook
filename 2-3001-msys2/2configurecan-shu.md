### Configure参数

* 标准选项参数：
  | --help | print this message | 显示此帮助信息 |
  | :--- | :--- | :--- |
  | --log\[=FILE\|yes\|no\] | log tests and output to FILE \[config.err\] | 记录测试并输出到config.err文件 |
  | --prefix=PREFIX | install in PREFIX \[/usr/local\] | 安装程序到指定目录 \[/usr/local\] |
